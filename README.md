## Requirements

* NodeJS V20+
* FFMPEG
* Imagemagick
* NodeJS basic knowledges (dont be an idiot)

#### Setup bot on your Machine
```bash
$ git clone https://github.com/ShirokamiRyzen/RyzumiMD-ESM
$ cd RyzumiMD-ESM
$ npm i
```

---------

## Run

```bash
npm start
```
